# AI Chat Frontend

Frontend of an AI chat application built using React (Vite).
This app allows users to send messages to an AI backend and recieve responses.

## Live Demo

https://chat-frontend-lake-six.vercel.app/

## Features
-Chat interface for AI conversation
-Loading indicator while AI is typing
-Error handling for failed requests
-Environment-based API configuration

## Tech Stack

-React(Vite)
-javaScript
-Fetch API

## Run locally

npm install
npm run dev

## Enviornment Variables

Create a `.env` file in the root:

VITE_API_URL=https://YOUR-BACKEND-URL.onrender.com

## Notes 

This frontend communicates with a separately deployed backend service
